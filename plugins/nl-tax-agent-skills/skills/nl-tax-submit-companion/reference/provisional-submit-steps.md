# Manual Submission Steps -- Voorlopige Aanslag 2026

## For REQUEST (aanvragen)

1. Go to Mijn Belastingdienst (mijn.belastingdienst.nl)
2. Log in with DigiD
3. Select "Voorlopige aanslag 2026 aanvragen"
4. Enter estimated income using the field map
5. Enter deductions and box 3 estimates
6. Review the summary
7. Submit the request
8. Expect a response (beschikking) within approximately 8 weeks

## For CHANGE (wijzigen)

1. Go to Mijn Belastingdienst (mijn.belastingdienst.nl)
2. Log in with DigiD
3. Select "Voorlopige aanslag 2026 wijzigen"
4. Enter ALL data again (not just the changes) using the field map
5. Review the summary
6. Submit the change
7. A new beschikking will be sent with adjusted monthly amounts

**Important:** When changing a voorlopige aanslag, you must re-enter all information, not just the fields that changed. Use the field map to ensure completeness.
