## Rule

These worked examples illustrate the box 3 calculation methods for tax year 2025. They demonstrate both the fictitious return (forfaitair rendement) and the actual return (werkelijk rendement) approaches and their comparison.

All percentage values used below are subject to confirmation: _verify against source snapshot -- percentages may be refined when definitive figures are published._

- Categorie I (banktegoeden): 0.36%
- Categorie II (overige bezittingen): 6.04%
- Categorie III (schulden): 2.47%
- Box 3 tax rate: 36%
- Heffingsvrij vermogen: EUR 57,000 per person

---

## Example 1: Simple savings only (single taxpayer)

### Facts

- Taxpayer: single, no fiscal partner
- Assets on 1 January 2025:
  - Savings account: EUR 80,000
- No investments, no debts
- Actual interest received in 2025: EUR 1,200

### Fictitious return calculation

| Step | Description | Amount |
|------|-------------|--------|
| 1 | Categorie I (banktegoeden) | EUR 80,000 |
| 2 | Categorie II (overige bezittingen) | EUR 0 |
| 3 | Categorie III (schulden) | EUR 0 |
| 4 | Total vermogen (I + II - III) | EUR 80,000 |
| 5 | Heffingsvrij vermogen | EUR 57,000 |
| 6 | Rendementsgrondslag (4 minus 5) | EUR 23,000 |
| 7 | Weighted fictitious return % | 0.36% (100% in Categorie I) |
| 8 | Forfaitair rendement (6 * 7) | EUR 82.80 |
| 9 | Box 3 tax (8 * 36%) | EUR 29.81 |

### Actual return calculation

| Description | Amount |
|-------------|--------|
| Interest received | EUR 1,200 |
| Total actual return | EUR 1,200 |

### Comparison

| Method | Taxable return | Box 3 tax |
|--------|---------------|-----------|
| Fictitious return | EUR 82.80 | EUR 29.81 |
| Actual return | EUR 1,200.00 | EUR 432.00 |

**Result:** The fictitious return (EUR 82.80) is lower than the actual return (EUR 1,200). The fictitious method is more favorable. The taxpayer would use the standard fictitious return method.

## Example 2: Mixed portfolio, actual return lower (single taxpayer)

### Facts

- Taxpayer: single, no fiscal partner
- Assets on 1 January 2025:
  - Savings account: EUR 30,000 (Categorie I)
  - Investment portfolio (listed securities): EUR 200,000 (Categorie II)
- Debts on 1 January 2025:
  - Personal loan: EUR 20,000 (Categorie III)
- Actual return in 2025:
  - Interest on savings: EUR 400
  - Dividends received: EUR 3,000
  - Unrealized loss on securities (mark-to-market): EUR -15,000
  - Custody fees: EUR 250

### Fictitious return calculation

| Step | Description | Amount |
|------|-------------|--------|
| 1 | Categorie I (banktegoeden) | EUR 30,000 |
| 2 | Categorie II (overige bezittingen) | EUR 200,000 |
| 3 | Categorie III (schulden) | EUR 20,000 |
| 4 | Total vermogen (I + II - III) | EUR 210,000 |
| 5 | Heffingsvrij vermogen | EUR 57,000 |
| 6 | Rendementsgrondslag (4 minus 5) | EUR 153,000 |

Weighted fictitious return percentage calculation:

| Category | Amount | Rate | Contribution |
|----------|--------|------|-------------|
| Categorie I | EUR 30,000 | 0.36% | EUR 108.00 |
| Categorie II | EUR 200,000 | 6.04% | EUR 12,080.00 |
| Categorie III | EUR 20,000 | 2.47% | EUR -494.00 (subtracted) |
| **Weighted total** | | | **EUR 11,694.00** |
| **Weighted % (11,694 / 210,000)** | | | **5.57%** |

| Step | Description | Amount |
|------|-------------|--------|
| 7 | Weighted fictitious return % | 5.57% |
| 8 | Forfaitair rendement (153,000 * 5.57%) | EUR 8,522.10 |
| 9 | Box 3 tax (8 * 36%) | EUR 3,067.96 |

### Actual return calculation

| Description | Amount |
|-------------|--------|
| Interest received | EUR 400 |
| Dividends received | EUR 3,000 |
| Unrealized loss on securities | EUR -15,000 |
| Deductible costs (custody fees) | EUR -250 |
| **Total actual return** | **EUR -11,850** |

### Comparison

| Method | Taxable return | Box 3 tax |
|--------|---------------|-----------|
| Fictitious return | EUR 8,522.10 | EUR 3,067.96 |
| Actual return | EUR -11,850 (negative) | EUR 0.00 |

**Result:** The actual return is negative (EUR -11,850) due to market losses. The actual return method results in EUR 0 box 3 tax, compared to EUR 3,067.96 under the fictitious method. The taxpayer should claim actual return. The workpack should present both calculations and note that the election is made in the official filing environment.

## Example 3: Fiscal partners with allocation optimization

### Facts

- Two fiscal partners (Partner A and Partner B)
- Combined assets on 1 January 2025:
  - Savings accounts: EUR 100,000 (Categorie I)
  - Investment portfolio: EUR 150,000 (Categorie II)
- Combined debts: EUR 10,000 (Categorie III)
- Combined heffingsvrij vermogen: EUR 114,000 (2 * EUR 57,000)

### Combined fictitious return calculation

| Step | Description | Amount |
|------|-------------|--------|
| 1 | Categorie I (banktegoeden) | EUR 100,000 |
| 2 | Categorie II (overige bezittingen) | EUR 150,000 |
| 3 | Categorie III (schulden) | EUR 10,000 |
| 4 | Total vermogen (I + II - III) | EUR 240,000 |
| 5 | Heffingsvrij vermogen (combined) | EUR 114,000 |
| 6 | Rendementsgrondslag (4 minus 5) | EUR 126,000 |

Weighted fictitious return percentage:

| Category | Amount | Rate | Contribution |
|----------|--------|------|-------------|
| Categorie I | EUR 100,000 | 0.36% | EUR 360.00 |
| Categorie II | EUR 150,000 | 6.04% | EUR 9,060.00 |
| Categorie III | EUR 10,000 | 2.47% | EUR -247.00 |
| **Weighted total** | | | **EUR 9,173.00** |
| **Weighted % (9,173 / 240,000)** | | | **3.82%** |

| Step | Description | Amount |
|------|-------------|--------|
| 7 | Weighted fictitious return % | 3.82% |
| 8 | Forfaitair rendement (126,000 * 3.82%) | EUR 4,813.20 |
| 9 | Combined box 3 tax (8 * 36%) | EUR 1,732.75 |

### Allocation scenarios

Partners can freely allocate box 3 assets and debts between them. The chosen ratio applies to the entire box 3 base (all categories). Below are three scenarios:

#### Scenario A: 50/50 split

| | Partner A | Partner B |
|---|----------|----------|
| Share of vermogen | EUR 120,000 | EUR 120,000 |
| Heffingsvrij vermogen | EUR 57,000 | EUR 57,000 |
| Rendementsgrondslag | EUR 63,000 | EUR 63,000 |
| Forfaitair rendement (3.82%) | EUR 2,406.60 | EUR 2,406.60 |
| Box 3 tax (36%) | EUR 866.38 | EUR 866.38 |
| **Combined tax** | | **EUR 1,732.75** |

#### Scenario B: 100/0 split (all to Partner A)

| | Partner A | Partner B |
|---|----------|----------|
| Share of vermogen | EUR 240,000 | EUR 0 |
| Heffingsvrij vermogen | EUR 57,000 | EUR 57,000 (unused) |
| Rendementsgrondslag | EUR 183,000 | EUR 0 |
| Forfaitair rendement (3.82%) | EUR 6,990.60 | EUR 0 |
| Box 3 tax (36%) | EUR 2,516.62 | EUR 0 |
| **Combined tax** | | **EUR 2,516.62** |

#### Scenario C: 75/25 split

| | Partner A | Partner B |
|---|----------|----------|
| Share of vermogen | EUR 180,000 | EUR 60,000 |
| Heffingsvrij vermogen | EUR 57,000 | EUR 57,000 |
| Rendementsgrondslag | EUR 123,000 | EUR 3,000 |
| Forfaitair rendement (3.82%) | EUR 4,698.60 | EUR 114.60 |
| Box 3 tax (36%) | EUR 1,691.50 | EUR 41.26 |
| **Combined tax** | | **EUR 1,732.75** |

### Allocation analysis

| Scenario | Allocation | Combined tax |
|----------|-----------|-------------|
| A | 50/50 | EUR 1,732.75 |
| B | 100/0 | EUR 2,516.62 |
| C | 75/25 | EUR 1,732.75 |

**Result:** For a flat-rate box 3 system with a single tax rate (36%), the allocation mainly matters when the combined vermogen is close to the combined heffingsvrij vermogen, or when partners have different marginal rates in other boxes that interact with thresholds. In this example, the 50/50 and 75/25 splits produce the same combined tax because both partners exceed their individual heffingsvrij vermogen. The 100/0 split is worse because Partner A's unused heffingsvrij vermogen portion is wasted (Partner B has nothing to shelter but retains an allowance).

The workpack should always compute multiple allocation scenarios and present the optimal split. When the total vermogen is just above the combined heffingsvrij vermogen, a skewed allocation that keeps one partner below their threshold can reduce the total tax.

## Developer instruction

When generating box 3 examples or performing calculations in the workpack:

1. Always show the full calculation chain: category totals, weighted percentage, rendementsgrondslag, forfaitair rendement, tax
2. For actual return comparisons, show both methods side by side with a clear recommendation note
3. For fiscal partners, compute at least three allocation scenarios (50/50, optimal, and one alternative)
4. Round to two decimal places for amounts and percentages
5. Always add the note that percentage values are subject to confirmation from the definitive source publication

## Common failure

Do not compute the weighted return percentage using the rendementsgrondslag (after heffingsvrij vermogen deduction). The weighted percentage is based on the gross category composition. The heffingsvrij vermogen is only deducted to determine the base on which the weighted percentage is applied.
