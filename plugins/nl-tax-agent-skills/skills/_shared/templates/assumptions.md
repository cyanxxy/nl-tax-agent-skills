# Assumptions Log

This file tracks assumptions made during workpack preparation. Every assumption must be reviewed before submission.

## Format

Each assumption entry should include:

- **ID:** assumption number (e.g., A001)
- **Category:** income / deduction / box3 / partner / own-home / other
- **Assumption:** what was assumed
- **Basis:** why this assumption was made (evidence, user statement, default)
- **Impact:** what changes if this assumption is wrong
- **Review required:** yes/no
- **Resolved:** yes/no

## Template

| ID | Category | Assumption | Basis | Impact | Review | Resolved |
|----|----------|-----------|-------|--------|--------|----------|
| A001 | example | Full-year Dutch resident | User statement | If incorrect, non-resident rules apply — out of v1 scope | Yes | No |

## Rules

1. Every estimated amount must have a corresponding assumption entry.
2. Partner allocation choices must be listed as assumptions until confirmed.
3. Assumptions from prior-year data extrapolation must be flagged.
4. No assumption may be silently resolved — all resolutions require human confirmation.
